### Key findings from ALAB exploration
- ALAB has both `render` and `proxy` purpose prims (17 proxy, 18 render in first 5000 prims)
- Proxy prims named like `proxyGeo` or `GEO_PROXY`
- Render prims named like `instancer` or `GEO`
- They share common parent prims (e.g. `proc_dev_circuit_board_large02_0003`)

### USD API used
- `UsdGeom.BBoxCache(time, ["default", "render", "proxy"]).ComputeWorldBound(prim)`
- `UsdGeom.Camera.GetCamera(time).frustum.ComputeViewMatrix() / ComputeProjectionMatrix()`
- Projection: 8 bbox corners × view-projection matrix → NDC → clamped → area / 4
